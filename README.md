This is a simplified banking system where a customer do one of the following options if applicable:

1) Deposit
2) Withdraw
3) Query Account
4) Transfer Funds
5) View last deposit amount
6) Cancel

There is also a Bank Staff Menu that can:

-Open an account
-Close an account
